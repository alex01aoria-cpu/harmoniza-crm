# Harmoniza CRM V1

Motor Operacional de Pré-venda e Conversão da Harmoniza.

## Objetivo

Construir a primeira versão funcional do CRM próprio da Harmoniza para:

- capturar origem rastreável de leads;
- organizar triagem e handoff comercial;
- controlar follow-ups;
- registrar perdas e vendas;
- oferecer leitura gerencial mínima;
- permitir automação e governança com Hermes Agent.

## Estrutura inicial

- `apps/api/` — backend FastAPI
- `apps/web/` — frontend Next.js
- `infra/docker/` — ambiente local e containers
- `docs/architecture/` — documentação técnica
- `docs/plans/` — planos de implementação
- `tests/e2e/` — fluxos críticos de ponta a ponta
